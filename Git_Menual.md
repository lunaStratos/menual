# Git 간단 설명

## 초기설정

1. git init

git 초기 등록

2. git remote add origin [address]

> ex: git remote add origin https://github.com/lunaStratos/menual.git



## commit & push

1. git add *

전체 대기열에 올리기

2. git commit -m "메시지"

메시지에 변경사항 적기

3. git push origin master

잘 안되면 -f를 추가하면 강제로 업로드 가능

> ex: git push origin master -f
